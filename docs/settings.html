Settings

To configure the operator, use the OperatorSettings that are configurable during the generic host extension method AddKubernetesOperator(IServiceCollection, Action<OperatorSettings>) or AddKubernetesOperator(IServiceCollection, OperatorSettings).

You can configure things like the name of the operator, if it should use namespacing, and other elements like the urls of metrics and lease durations for the leader election.

Please look at the documention over at: OperatorSettings to know what the fields mean.
