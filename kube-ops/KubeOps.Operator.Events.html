Namespace KubeOps.Operator.Events

Interfaces

IEventManager

Event manager for k8s.Models.Corev1Event objects. Contains various utility methods for emitting events on objects.

Enums

EventType

The type of a k8s.Models.Corev1Event. The event type will be stringified and used as k8s.Models.Corev1Event.Type.

Delegates

IEventManager.Publisher

Delegate that publishes a predefined event for a resource that is passed. This delegate should be created with CreatePublisher(String, String, EventType). When called with a resource, the publisher creates or updates the event defined by the params.

IEventManager.StaticPublisher

Delegate that publishes a predefined event for a statically given resource. This delegate should be created with CreatePublisher(IKubernetesObject<V1ObjectMeta>, String, String, EventType). When called, the publisher creates or updates the event defined by the params.
